# MLB.com - Manual Test Cases

This document contains the manual test cases designed for MLB.com as part of a QA/Test Engineering portfolio aimed at applying to the MLB Technology Internship.

Each case includes: **ID, flow, preconditions, steps, expected result, and priority.**

---
